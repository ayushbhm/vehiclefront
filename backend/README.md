# Vehicle Parking App

This is a simple web application to manage parking spots. It uses Flask and SQLAlchemy.

## Setup

1. Create a virtual environment: `python3 -m venv venv`
2. Activate the virtual environment: `source venv/bin/activate` (on Linux/Mac) or `venv\Scripts\activate` (on Windows)
3. Install the requirements: `pip install -r requirements.txt`
4. Run the application: `python app.py`

## Usage

1. Go to `http://localhost:5000` in your browser
2. Register as a user
3. Book a parking spot
4. View your reservations
5. Release a parking spot
6. View your parking history

7. Login as an admin with `username: admin` and `password: admin123`
